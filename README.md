# EduPulse-Smart-Campus-Feedback-System
EduPulse is a web-based application that allows students to submit feedback about campus facilities and courses developed using HTML, CSS, JSP and MySQL

## Technologies Used
- HTML
- CSS
- JSP
- MySQL

## Features
- Student login system
- Submit campus and course feedback
- Feedback stored in MySQL database
- Faculty/Admin can view feedback
- Simple and responsive user interface

## Project Description
Developed a web-based system for students to submit campus and course feedback.
Implemented JSP and MySQL to store and manage feedback data.
Designed user interfaces using HTML and CSS.
